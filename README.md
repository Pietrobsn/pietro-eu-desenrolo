# Pietro eu desenrolo

Landing pública de serviços do Pietro, com atendimento direto pelo WhatsApp para quem precisa resolver uma bronca simples de tecnologia sem conversa complicada.

## Sobre o site

O site apresenta serviços práticos para criação de ferramentas simples, automações, instalação de programas e suporte técnico. A proposta é manter uma comunicação direta, jovem e fácil de entender, sem parecer uma empresa gigante falsa.

## Serviços oferecidos

- Criação de sistemas simples
- Automações
- Instalação e configuração de programas
- Suporte técnico
- Manutenção e configuração de PC
- Windows, Office, drivers e ajustes
- Ajuda com computador dando problema

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript
- Dicionário local para idioma Português do Brasil / Italiano
- LocalStorage para salvar preferência de idioma

## Como rodar localmente

Abra o arquivo `index.html` no navegador ou rode um servidor local simples:

```bash
python -m http.server 4175 --bind 127.0.0.1
```

Depois acesse:

```text
http://127.0.0.1:4175/
```

## Idiomas

O site tem suporte a:

- Português do Brasil, idioma padrão
- Italiano

A troca é feita pelo seletor `PT / IT` no topo do site, sem Google Translate e sem API externa. A preferência fica salva no navegador usando `localStorage`.
